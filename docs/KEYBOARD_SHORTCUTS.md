# CircleOS Keyboard Shortcuts

## Desktop Navigation

| Shortcut | Action |
|----------|--------|
| `Super` | Open application launcher |
| `Super + L` | Lock screen |
| `Super + D` | Show desktop |
| `Super + Tab` | Switch windows |
| `Super + Q` | Close window |
| `Super + F` | Toggle fullscreen |
| `Super + M` | Minimize window |
| `Alt + Tab` | Cycle through windows |
| `Alt + F4` | Close window |

## System Controls

| Shortcut | Action |
|----------|--------|
| `Super + S` | Open settings |
| `Super + T` | Open terminal |
| `Super + E` | Open file manager |
| `Super + N` | Open notifications |
| `Super + P` | Power menu |
| `Ctrl + Alt + Delete` | System monitor |

## Window Management

| Shortcut | Action |
|----------|--------|
| `Super + Left` | Snap window left |
| `Super + Right` | Snap window right |
| `Super + Up` | Maximize window |
| `Super + Down` | Restore/minimize window |
| `Super + 1-9` | Switch to workspace 1-9 |
| `Super + Shift + 1-9` | Move window to workspace 1-9 |

## Media Controls

| Shortcut | Action |
|----------|--------|
| `XF86AudioRaiseVolume` | Increase volume |
| `XF86AudioLowerVolume` | Decrease volume |
| `XF86AudioMute` | Toggle mute |
| `XF86AudioPlay` | Play/pause media |
| `XF86AudioNext` | Next track |
| `XF86AudioPrev` | Previous track |

## Screenshots

| Shortcut | Action |
|----------|--------|
| `Print` | Screenshot entire screen |
| `Alt + Print` | Screenshot current window |
| `Shift + Print` | Screenshot selection |
| `Ctrl + Print` | Screenshot to clipboard |

## Accessibility

| Shortcut | Action |
|----------|--------|
| `Super + Alt + S` | Toggle screen reader |
| `Super + Alt + H` | Toggle high contrast |
| `Super + Alt + +` | Zoom in |
| `Super + Alt + -` | Zoom out |
| `Super + Alt + 0` | Reset zoom |

## Terminal

| Shortcut | Action |
|----------|--------|
| `Ctrl + Alt + T` | Open terminal |
| `Ctrl + Shift + C` | Copy (in terminal) |
| `Ctrl + Shift + V` | Paste (in terminal) |
| `Ctrl + Shift + T` | New terminal tab |
| `Ctrl + Shift + W` | Close terminal tab |

## File Manager

| Shortcut | Action |
|----------|--------|
| `Ctrl + N` | New window |
| `Ctrl + T` | New tab |
| `Ctrl + W` | Close tab |
| `Ctrl + H` | Toggle hidden files |
| `Ctrl + L` | Focus location bar |
| `Alt + Up` | Parent directory |
| `Alt + Left` | Back |
| `Alt + Right` | Forward |
| `F2` | Rename |
| `Delete` | Move to trash |
| `Shift + Delete` | Delete permanently |

## Customization

Keyboard shortcuts can be customized in:
**Settings → Keyboard → Shortcuts**

## Tips

- Hold `Super` key to see all shortcuts overlay
- Most shortcuts work with `Ctrl` on non-Super keyboards
- Custom shortcuts can be added for any application
- Shortcuts are context-aware (work differently in different apps)
